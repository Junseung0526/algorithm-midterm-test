# 그래프 탐욕 알고리즘 (Graph Greedy Algorithm)

## 1. 개요
그래프 구조에서 최소 비용 신장 트리(MST)를 찾거나, 단일 출발점에서의 최단 경로를 구하는 데 탐욕적 기법이 효과적으로 사용됩니다.

## 2. 주요 알고리즘 분석

### 2.1. 크루스칼 알고리즘 (Kruskal's Algorithm)
- **방식**: 간선 중심. 모든 간선을 가중치 순으로 정렬한 뒤, 사이클을 만들지 않는 간선을 순차적으로 선택함.
- **시간 복잡도**: $O(E \log E)$
- **시험 포인트**: 
    - **Union-Find** 자료구조를 사용하여 사이클 판별.
    - 간선이 적은 희소 그래프(Sparse Graph)에 유리.

### 2.2. 프림 알고리즘 (Prim's Algorithm)
- **방식**: 정점 중심. 시작점에서부터 연결된 간선 중 가장 가중치가 작은 정점을 선택하며 트리를 확장함.
- **시간 복잡도**: $O(E \log V)$ (우선순위 큐 사용 시)
- **시험 포인트**: 
    - 매 단계마다 **우선순위 큐**를 갱신하여 최적의 간선 선택.
    - 간선이 많은 밀집 그래프(Dense Graph)에 유리.

### 2.3. 다익스트라 알고리즘 (Dijkstra's Algorithm)
- **방식**: 최단 경로 탐색. 출발 노드에서 거리가 가장 짧은 노드를 선택하고, 그 노드를 거쳐가는 경로를 갱신(Relaxation)함.
- **시간 복잡도**: $O(E \log V)$
- **시험 포인트**: 
    - **음수 가중치**가 있는 그래프에서는 동작하지 않음 (그리디의 한계).
    - 매번 '현재 확정된 최단 거리'를 기준으로 탐욕적 선택 수행.

---

## 3. 그래프 알고리즘 비교 (요약)

| 알고리즘 | 목적 | 중심 요소 | 시간 복잡도 | 핵심 자료구조 |
| :--- | :--- | :--- | :--- | :--- |
| **크루스칼** | MST 찾기 | 간선 (Edge) | $O(E \log E)$ | Union-Find |
| **프림** | MST 찾기 | 정점 (Vertex) | $O(E \log V)$ | Priority Queue |
| **다익스트라** | 최단 경로 | 정점 (Vertex) | $O(E \log V)$ | Priority Queue |
