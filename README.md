# 알고리즘 중간고사 완벽 대비 (Algorithm Midterm Test)

이 레포지토리는 알고리즘 중간고사를 대비하여 주요 알고리즘의 핵심 개념, 파이썬 구현, 그리고 시험 꿀팁을 정리한 공간입니다.


---

## 📚 수록된 알고리즘 설명

각 파일에는 **시험 꿀팁**과 **코드 진행 순서**가 상세히 주석으로 달려 있습니다.

1.  **[Greedy.py](./Greedy.py) (탐욕 알고리즘)**
    *   매 순간 최적의 선택을 하는 방식. 배낭 문제(Fractional) 예제 포함.
2.  **[MergeSort.py](./MergeSort.py) (합병 정렬)**
    *   분할 정복(Divide & Conquer)의 대표 사례. 언제나 $O(n \log n)$ 보장.
3.  **[Backtracking.py](./Backtracking.py) (백트랙킹)**
    *   유망하지 않은 경로는 가지치기(Pruning)하며 탐색. N-Queens 예제.
4.  **[BranchAndBound.py](./BranchAndBound.py) (분기 한정)**
    *   최상 우선 탐색(Best-First) 기반 최적화 문제 해결. 0/1 배낭 문제.
5.  **[Search.py](./Search.py) (탐색 알고리즘)**
    *   정렬된 데이터에서 로그 시간 내에 찾는 이진 탐색(Binary Search).
6.  **[Sort.py](./Sort.py) (정렬 알고리즘)**
    *   평균적으로 매우 빠른 성능을 내는 퀵 정렬(Quick Sort).
7.  **[Approximation.py](./Approximation.py) (근사 알고리즘)**
    *   NP-Hard 문제에 대해 최적해에 가까운 답을 빠르게 도출.
8.  **[Parallel.py](./Parallel.py) (병렬 알고리즘)**
    *   멀티 코어를 활용하여 여러 작업을 동시에 처리.
9.  **[Distributed.py](./Distributed.py) (분산 알고리즘)**
    *   독립된 노드들이 네트워크를 통해 협력하는 마스터-워커 모델.
10. **[Quantum.py](./Quantum.py) (양자 알고리즘)**
    *   큐비트의 중첩과 측정 개념을 활용한 차세대 알고리즘 시뮬레이션.

---

## 💡 학습 방법
1. 각 `.py` 파일을 열어 상단의 **[시험 꿀팁!]** 을 암기합니다.
2. 코드 내의 **알고리즘 진행:** 주석을 따라가며 로직을 이해합니다.
3. 직접 코드를 실행해 보며 결과가 어떻게 도출되는지 확인합니다.

열공해서 중간고사 대박 나세요! 🔥
